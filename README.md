# New SS Events — Premium Event Management

New SS Events is a high-end, multi-page event management website built with **React**, **Vite**, and **Framer Motion**. It features a luxurious jewel-toned design, smooth scroll animations, and a fully responsive layout tailored for premium event services in Nalgonda.

## ✨ Features

- **Premium Design System**: Sophisticated color palette (Deep Plum, Gold, Ivory) and elegant typography (Playfair Display & Inter).
- **Smooth Animations**: Page transitions and scroll-reveal effects powered by Framer Motion.
- **Responsive Layout**: Seamless experience across mobile, tablet, and desktop devices.
- **Pages**:
  - **Home**: Dynamic hero section and service highlights.
  - **About**: Company mission, values, and team.
  - **Services**: Detailed service categories and transparent pricing.
  - **Gallery**: Filterable portfolio with interactive lightbox.
  - **Contact**: Functional lead generation form and real-time contact details.

## 🛠️ Tech Stack

- **Frontend**: React (Vite)
- **Styling**: Vanilla CSS (Modern CSS Variables & Grid/Flexbox)
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Routing**: React Router DOM

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v16 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/JagadeeshO11/NewSSEvents.git
   ```
2. Navigate to the project directory:
   ```bash
   cd NewSSEvents
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
